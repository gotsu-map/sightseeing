# GitHub Pages公開用ファイル

この `docs/` フォルダをGitHub Pagesの公開元に設定してください。

- `index.html`: トップページ
- `styles.css`: デザイン
- `app.js`: コース作成・レビュー保存の動作
- `cards.json`: 観光カードデータ
- `assets/`: 画像素材

GitHub Pages設定:

1. `Settings` → `Pages`
2. `Source`: `Deploy from a branch`
3. `Branch`: `main`
4. Folder: `/docs`
